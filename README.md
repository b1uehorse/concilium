# Council CLI — Мост между Claude Code и Gemini CLI

Утилита для проведения "консилиума" нейросетей через буфер обмена.

## Установка

Скрипт должен находиться в вашем `PATH`. Рекомендуется `~/bin/council`.

```bash
chmod +x ~/bin/council
```

## Использование

### Базовые роли
- `council review` (или просто `council`) — код-ревью.
- `council alt` — поиск альтернативных решений.
- `council devil` — роль адвоката дьявола (поиск уязвимостей и edge cases).
- `council sql` — проверка SQL и запросов на производительность (N+1, индексы).
- `council sec` — аудит безопасности.

### Команды
- `council log` — просмотр истории сессии (`~/.council/session.md`).
- `council clear` — очистка логов и сброс счетчика раундов.
- `council help` — справка.

### Режим Loop
Запускает серию раундов. После каждого ответа Gemini скрипт копирует его в буфер обмена и ждет нажатия Enter, чтобы прочитать новый ответ из CC из буфера.
```bash
council loop 5 review
```

## Как это работает
1. Скрипт берет текст из `stdin` или (если `stdin` пуст) из системного буфера обмена (`pbpaste` на macOS).
2. Оборачивает его в системный промпт выбранной роли.
3. Отправляет запрос в `gemini -p` (headless mode).
4. Полученный ответ форматирует для вставки в Claude Code и кладет обратно в буфер обмена (`pbcopy`).
5. Все раунды логируются в `~/.council/session.md`.
